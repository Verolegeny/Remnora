# Remnora

A text-based survival RPG written in Python.

## Features

- Multiple story paths
- HP system
- Inventory system
- Multiple endings
- Choice-based gameplay

## Technologies

- Python

## How to Run

```bash
python Remnora.py
```

## About

This is my first Python project while learning backend development fundamentals.

The goal of the project was to practice:
- if/else logic
- while loops
- game flow
- variables
- inventory handling
- branching story paths
